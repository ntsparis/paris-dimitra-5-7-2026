# Παρασκευάς & Δήμητρα — 5 Ιουλίου 2026

A single-page Greek wedding invitation website, built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools.

## Live Sections

- **Hero** — names, date, animated entrance, parallax background
- **Countdown** — live ticker to the wedding datetime
- **Our Story** — milestone cards (2020 → 2026)
- **Families** — parents of bride and groom
- **Κουμπάρα** — Δέσποινα Τόκα
- **Program** — day-of timeline
- **Ceremony** — Ιερός Ναός Προφήτη Ηλία, 19:50, embedded map
- **Reception** — Κτήμα Έλενα, Αμπελώνας, ~21:00, embedded map
- **Q&A** — frequently asked questions
- **Footer**

## Design

Warm, serif-heavy typographic system using **Cormorant Garamond**, **EB Garamond**, and **Cinzel** from Google Fonts. Color palette of corals, blushes, sages, and creams. Scroll-triggered entrance animations via `IntersectionObserver`.

## Development

No install step. Open `index.html` directly in a browser.

```bash
open index.html
```
